# Excel Sales Analytics Dashboard

## Project Overview
An interactive sales dashboard built in Excel using Pivot Tables, Advanced Charts, and Timelines to analyze $6.7M in revenue data.

## Features & Workflow
1. **Data Cleaning (`cleaner.py`):** Automated raw data preprocessing using Python to handle missing records, filter anomalies, and structure fields for analysis.
2. **Database Management (SQL):** Structured and queried records to extract relevant financial datasets.
3. **Data Visualization (`cleaned_sales_data.xlsb`):** Designed a dynamic, user-friendly Excel dashboard utilizing Pivot Tables and Slicers to track key performance indicators (KPIs) and monthly revenue trends.

## Tech Stack
* **Language:** Python
* **Database:** SQL
* **Analytics & Visualization:** Microsoft Excel (Pivot Tables, Advanced Formatting, Timelines)
